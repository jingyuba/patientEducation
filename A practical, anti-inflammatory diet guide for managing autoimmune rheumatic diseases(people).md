# **自體免疫風濕病飲食建議實踐指南**

### **前言**

本指南旨在為類風濕性關節炎（RA）、全身性紅斑狼瘡（SLE）及修格蘭氏症（乾燥症，SS）的患者，提供一份以科學研究為基礎的日常飲食實踐策略。飲食調整是輔助正規醫療的重要環節，目的在於幫助控制發炎、管理症狀並提升整體生活品質。**請注意，任何飲食調整都不能取代醫師處方的藥物治療**，在進行重大改變前，請務必諮詢您的醫師或註冊營養師。

### **核心飲食原則：以「地中海飲食」為藍本**

地中海飲食模式是目前科學證據最充分、最被推薦的抗發炎飲食策略。其核心並非嚴格限制，而是強調選擇天然、完整的食物。

#### **應優先攝取的食物**

* **大量的蔬菜與水果：** 每日攝取多種顏色的蔬果，以獲得豐富的抗氧化物、維生素和多酚 。  
* **全穀類與豆類：** 以糙米、燕麥、藜麥等全穀物取代精緻澱粉；多攝取扁豆、鷹嘴豆等豆類，其富含的膳食纖維有助於維持腸道健康 。  
* **優質脂肪來源：**  
  * **橄欖油：** 使用特級初榨橄欖油作為主要烹飪用油 。  
  * **堅果與種子：** 如核桃、杏仁、亞麻籽等，是健康的脂肪和纖維來源。  
* **高脂魚類：** 每週至少食用兩次富含Omega-3脂肪酸的魚類，如鮭魚、鯖魚、沙丁魚 。  
* **天然香料：** 多加利用薑黃、生薑、大蒜、肉桂等具有抗發炎潛力的香料 。

#### **應限制攝取的食物**

* **紅肉與加工肉品：** 限制牛肉、豬肉的攝取量，並盡量避免香腸、火腿等加工肉品 。  
* **精緻碳水化合物與添加糖：** 減少白麵包、白米、糕點、含糖飲料及甜食的攝取 。  
* **不健康的脂肪：** 避免油炸食物、人造奶油及含有飽和脂肪與反式脂肪的加工零食 。  
* **高鹽分食品：** 減少加工食品、罐頭及重調味醬料的攝取，以控制鈉含量 。

### **針對不同疾病的飲食策略重點**

#### **類風濕性關節炎 (Rheumatoid Arthritis, RA)**

* **首要建議：** 嚴格遵循地中海飲食模式，研究證實有助於改善疾病活動度、減輕疼痛及改善身體功能 。  
* **Omega-3補充：** 作為飲食的輔助，可考慮每日補充約2克的Omega-3脂肪酸（魚油），有助於緩解疼痛與晨僵 。  
* **植物性飲食：** 純素或素食飲食可能對部分患者的疼痛有所緩解，但證據等級不如地中海飲食明確。若採行需特別注意營養均衡，避免缺乏維生素B12、鐵質等營養素 。

#### **全身性紅斑狼瘡 (Systemic Lupus Erythematosus, SLE)**

* **體重與疲勞管理：** 對於體重超重的患者，採取低熱量或低升糖指數（GI）飲食，有助於減重並顯著改善疲勞感，且不會引發疾病發作 。  
* **心血管保護：** 地中海飲食因其對心血管的益處，是SLE患者的理想選擇 。  
* **腎臟照護：** 若有腎臟相關併發症，應嚴格限制鈉的攝取量（建議每日少於3克）。  
* **關鍵營養素補充：**  
  * **維生素D：** SLE患者普遍存在維生素D缺乏，且其水平與疾病活動度呈負相關。應定期檢測並依醫師建議進行補充 。  
  * **Omega-3脂肪酸：** 無論是透過攝取高脂魚類或亞麻籽油（每日30克），皆有助於降低發炎指標及改善腎功能 。

#### **修格蘭氏症／乾燥症 (Sjögren's Syndrome, SS)**

* **改善乾燥症狀：** 遵循地中海飲食，特別是攝取足夠的魚類和堅果，研究發現與眼部乾燥症狀的嚴重程度降低有關 。  
* **脂肪攝取注意：** 動物研究顯示，高飽和脂肪的飲食可能加劇腺體發炎並減少唾液分泌，因此建議限制此類脂肪的攝取 。  
* **飲食模式總結：** 雖然針對乾燥症的飲食研究較少，但採納全面的抗發炎飲食模式（如地中海飲食）是目前最合理的策略 。

### **實用烹飪與生活建議**

* **選擇健康的烹飪方式：** 多採用蒸、煮、燉的方式，限制燒烤、油炸和高溫烘烤，以減少在烹飪過程中產生的促發炎物質「糖化終產物」（AGEs）。  
* **維持健康體重：** 過重會增加關節負擔及全身性發炎，透過健康飲食和適度運動維持理想體重至關重要。
